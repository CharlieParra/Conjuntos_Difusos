Conjuntos Difusos

Un conjunto clsico es un conjunto con un lmite ntido. Por ejemplo, un conjunto clsico de nmeros reales mayores que 6 puede expresarse como A={xx>6}, donde hay un lmite claro e inequvoco en 6, de manera que si x es mayor que este nmero, entonces x pertenece al conjunto A; de lo contrario, x no pertenece al conjunto A. Aunque los conjuntos clsicos son adecuados para varias aplicaciones y han demostrado ser una herramienta importante para las matemticas y la informtica, no reflejan la naturaleza de los conceptos y pensamientos humanos, que tienden a ser abstractos e imprecisos. Como ilustracin, matemticamente podemos expresar el conjunto de todas las personas con una altura superior a 6 pies; este es el conjunto denotado por la Ecuacin (2.1), si dejamos que A = "persona alta" y x = "altura." Sin embargo, esta es una forma antinatural e inadecuada de representar nuestro concepto habitual de "persona alta." Por un lado, la naturaleza dicotmica del conjunto clsico clasificara a una persona de 6.001 pies de altura como una persona alta, pero no a una persona de 5.999 pies de altura. Esta distincin es intuitivamente irrazonable. El defecto proviene de la transicin abrupta entre la inclusin y la exclusin en un conjunto.

En contraste con un conjunto clsico, un conjunto difuso, como su nombre lo indica, es un conjunto sin un lmite ntido. Es decir, la transicin de "pertenecer a un conjunto" a "no pertenecer a un conjunto" es gradual, y esta transicin suave se caracteriza por funciones de pertenencia que dan a los conjuntos difusos flexibilidad en la modelacin de expresiones lingsticas comnmente utilizadas como "el agua est caliente" o "la temperatura es alta." Como seal Zadeh en 1965 en su artculo seminal titulado "Conjuntos Difusos," estos conjuntos o clases definidos de manera imprecisa "juegan un papel importante en el pensamiento humano, particularmente en los dominios de reconocimiento de patrones, comunicacin de informacin y abstraccin." Cabe destacar que la difusividad no proviene del azar de los miembros constituyentes de los conjuntos, sino de la naturaleza incierta e imprecisa de los pensamientos y conceptos abstractos.

Funciones de membresa

Una de las piezas fundamentales en la teora de conjuntos difusos es la funcin de membresa. Esta funcin, generalmente denotada como (), mide el grado de pertenencia de un elemento al conjunto difuso . A diferencia de los conjuntos clsicos, donde la funcin de pertenencia slo toma los valores 0 o 1, la funcin de membresa en un conjunto difuso puede tomar cualquier valor real en el intervalo [0, 1]. Un valor de 0 indica que el elemento no pertenece al conjunto, un valor de 1 indica plena pertenencia, y cualquier valor intermedio indica un grado parcial de pertenencia.

import matplotlib.pyplot as plt
import numpy as np

def MF_triangularIzquierda(X, a, b):
    if X <= a:
        return 1
    elif a < X < b:
        return -(1 / (b - a)) * (X - a) + 1
    else:
        return 0

def MF_trapezoidal(X, a, b, c, d):
    if X <= a:
        return 0
    elif a < X <= b:
        return (X - a) / (b - a)
    elif b < X <= c:
        return 1
    elif c < X < d:
        return (d - X) / (d - c)
    else:
        return 0

Funcin para evaluar una funcin de membresa izquierda-derecha (L-R)

$$ \text{LR}(x; c, \alpha, \beta) = \begin{cases} F_L\left(\frac{c - x}{\alpha}\right), & x \leq c. \\ F_R\left(\frac{x - c}{\beta}\right), & x \geq c. \end{cases} $$
$$ \begin{aligned} F_L(x) &= \max(0, \sqrt{1 - x^2}). \\ F_R(x) &= e^{-|x|^3}. \end{aligned} $$
  • x = Punto de evaluacin.
  • $\alpha$ = Define la dispersin de la parte izquierda de la funcin.
  • $\beta$ = Define la dispersin de la parte derecha de la funcin.
  • c = Valor del centro de la funcin.

def MF_leftRight(X, alpha, beta, c):
    if X >= c:
        x1 = (X - c) / beta
        return np.exp(-abs(x1**3))
    else:
        x2 = (c - X) / alpha
        return max(0, np.sqrt(1 - x2**2))
